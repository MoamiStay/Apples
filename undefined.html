Universal selector:

The CSS Universal Selector (*) selects all HTML elements on the page. 

        * { padding: 0; margin: 0; }

Instead of writing declarations for each selector individually you can separate each selector with a comma: 

       h1, h2, h3 {text-align: center; color: Maroon; }

PSEUDO-CLASSES!

pseudo-classes are used to:

  • Style an element whien a user mouses over it
  • Style visited and unvisited links differently
  • Style an element when it gets focus

Syntax:

        selector:pseudo-class {
          property: value;
        }

Different ways of displaying links with actions:

          a:link { color: yellow; }
        
          a:visited { color: red; }
        
          a:hover { color: white; }
        
          a:active { color: pink; }
Visit Moami_Stay!

Note!

a:hover MUST come after a:link and a:visited in the CSS definition in order to be effective!

Likewise a:active MUST come after a:hover ! Psuedo-class names are not case-sensitive.

Also..

Pseudo-classes can be combined with CSS classes:

          a.highlight:hover {
            color: red;
          }

Pseudo-classes can also be used on other elements like the <div> element: 

            div:hover {
              background-color: green;
            }
